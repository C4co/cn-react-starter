### Layers

```
  📁 /src
    📁 /assets (images, videos, etc...)
    📁 /components (basic components)
    📁 /pages (pages of features)
    📁 /routes (config about routes)
    📁 /services (server communication)
    📁 /templates (page layouts)
    📁 /values (constant values)
```

### Commands

**Start**
```
yarn start
```

**Tests**
```
yarn test
yarn test:once
```

**Build**
```
yarn build
```

**Formating**
```
yarn format
```

**Render**
```
yarn render
```
